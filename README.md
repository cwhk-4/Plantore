ゲーム概要
----
動物とその動物が住む環境を最適化するとクリアできるゲームです。

最適化を行うには、ミッションをクリアする必要があります。

そのミッションは草や木などのアイテムをユーザーが配置や整理する必要があるもので、クリアすると新しいアイテムや新しい動物が増えます。

ミッションをクリアしながら、全部のアイテムと動物を発見しましたらゲームクリアです。
